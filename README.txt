Foi utilizada a API Graphviewer disponibilizadas pelos docentes da unidade curricular. É necessário incluir o ficheiro Graphviewer.jar no sistema de ficheiros. É de notar que nesta, a macro 'linux' foi mudada para '__linux__', de modo a poder ser compilada num sistema operativo Ubuntu.
Foi utilizado o compilador g++ para Linux e Windows (testado e funcional em ambos os sistemas operativos).
Os ficheiros de input utilizados foram os dos mapas disponibilizados pelos monitores da unidade curricular. Encontram-se na pasta "Maps" dentro da pasta source ("src").
O projeto foi desenvolvido no IDE Eclipse.
